# `apps/global-page`

Vite + React + Tailwind UI for the Forge Jira global page. Production build output goes to `../main/build/global-page` (see `vite.config.ts`).

See the [repository README](../../README.md) for full setup, tunnel + port 3000, and build commands.

**Dev (Vite only):** from the repo root, `bun run dev --filter=global-page`.
