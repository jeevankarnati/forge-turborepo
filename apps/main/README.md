# `apps/main`

Atlassian Forge app: `manifest.yml`, tRPC resolver, and server-side code. The Jira global page static assets are built by `apps/global-page` into `build/global-page`.

See the [repository README](../../README.md) for install, `forge tunnel` / deploy, and Turborepo filters.

**Dev (Forge tunnel only):** from the repo root, `bun run dev --filter=main`.
